# Pomoland (뽀모도로 땅따먹기) — Frontend

> 공부 시간을 보여주는 서비스가 아니라, 공부한 시간이 내 땅이 되게 한다.

뽀모도로 세션을 완주할 때마다 포인트를 얻고, 그 포인트로 모두가 공유하는 맵 위의 땅(타일)을 점령·방어하는 게임형 학습 동기 부여 서비스의 웹 프론트엔드입니다.

## 기획 배경

뽀모도로 기법(집중 25분 + 휴식 5분 반복)은 널리 쓰이지만, 타이머 자체는 "공부를 시작하게 만들고 25분을 끝까지 채우게 만드는 동기"까지는 주지 못합니다. 혼자 공부하는 사용자는 다음과 같은 문제를 겪습니다.

- 타이머를 켜는 것 자체가 순전히 개인 의지에 달려 있다.
- 세션을 중간에 끊어도 아무 손해가 없어 완주할 이유가 약하다.
- 공부 시간이 숫자 기록으로만 남아 시각적인 성취감이 약하다.
- 경쟁·상호작용 요소가 없어 며칠 쓰다 흥미를 잃는다.

기존 서비스는 "내 기록을 남기고 남과 숫자로 비교"하거나(열품타류), "내 개인 공간에 보상을 쌓는"(Forest류) 구조입니다. Pomoland는 세션 **"완주"에만 보상을 지급**해 게임 규칙 자체가 뽀모도로 원칙(끊지 않는 집중)을 강제하도록 설계했습니다. 공부 시간이 숫자가 아니라 모두가 공유하는 맵 위의 눈에 보이는 영토로 남고, 다른 사용자와 타일을 점령·방어·탈환하며 직접 상호작용한다는 점이 차별점입니다.

## 게임 규칙 (서버 판정 기준)

모든 판정은 클라이언트가 아니라 서버가 기록한 세션 데이터를 기준으로 계산되며, 프론트엔드는 이를 화면에 반영하는 역할을 합니다.

- **세션**: 유저별 `studyTime`(기본 25분) 길이의 세션을 생성하고, 30초 주기로 하트비트를 보내 진행 중임을 알립니다. 하트비트가 끊기면 세션은 만료 처리되며, 동시에 살아있는 세션은 유저당 1개만 허용됩니다.
- **완주 판정**: 세션 종료 시각이 지난 뒤에만 완료 처리가 가능하고, 포기(abandon)한 세션은 보상이 없습니다.
- **포인트 적립**: 세션 완료 시 `(세션 분 / 5) * 2` 포인트를 지급합니다 (25분 세션 기준 10포인트). 동시에 일자별 공부시간 통계가 갱신되어 랭킹/통계에 반영됩니다.
- **타일 점령(occupy)**: 빈 타일은 투자한 포인트만큼 방어력으로 설정되어 그대로 점령되고, 이미 점유된 타일은 현재 방어력보다 큰 포인트를 투자해야 빼앗을 수 있습니다. 스폰 포인트 타일은 점령 불가하며, 내가 이미 보유한 타일과 상하좌우로 인접한 좌표만 점령할 수 있습니다.
- **타일 방어(defense)**: 내가 보유한 타일에 포인트를 추가로 투자해 방어력을 높일 수 있습니다.
- **스폰 포인트**: 맵 범위 내 좌표를 지정해 자신의 첫 거점을 만듭니다.
- **랭킹**: 일간/주간 공부시간(KST 기준), 보유 타일 수, 보유 포인트 4가지 기준으로 상위 N명을 조회합니다.

## 주요 화면/기능

- **소셜 로그인**: Apple, Google OAuth2 로그인 및 JWT(Access/Refresh) 기반 인증
- **뽀모도로 타이머**: 세션 생성 → 하트비트로 진행 유지 → 완료/포기, 완주 시 포인트·공부시간 자동 반영
- **맵 & 타일**: 전체 맵/타일 조회, 타일 점령, 방어력 강화, 스폰 포인트 지정
- **랭킹**: 일간/주간 공부 시간, 보유 타일 수, 보유 포인트 랭킹 조회
- **유저 설정**: 내 정보 조회/수정, 회원 탈퇴, 공부/휴식 시간 등 개인 설정

## 기술 스택

- **Language**: TypeScript
- **Framework/Library**: React 19, React Router
- **Build Tool**: Vite
- **연동 Backend**: Spring Boot 4.1 + MariaDB (별도 저장소)

인증이 필요한 API는 JWT Access Token을 `Authorization` 헤더로 전달해야 합니다. 맵 갱신은 MVP 기준 주기적 폴링 방식입니다(추후 WebSocket 기반 실시간 갱신으로 확장 예정). 상세 엔드포인트는 백엔드 저장소 README를 참고하세요.

## 로드맵

- 방어력 자동 감소(decay), 신규 유저 스폰 주변 보호 구역 등 밸런스/부정행위 방지 로직
- 시즌제(주기적 맵 리셋)와 시즌 보상(뱃지, 타일 테마)
- WebSocket(STOMP) 기반 실시간 맵 갱신
- 잔디(연속 공부 일수) 시각화, 과목/태그별 통계
- 탭 이탈·화면 전환 감지를 통한 부정행위 방지
